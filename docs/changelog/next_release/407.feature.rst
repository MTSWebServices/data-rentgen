Add ``GET /v1/jobs/dependencies`` API endpoint to retrieve job dependency graph (ancestors and descendants) for a given job.
Supports ``direction`` parameter with values ``UPSTREAM``, ``DOWNSTREAM``, or ``BOTH`` (default).
